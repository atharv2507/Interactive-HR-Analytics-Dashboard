# 📊 HR Analytics Dashboard

## 🔹 Overview
This project is an interactive HR dashboard built using Tableau to analyze employee attrition, job satisfaction, and workforce trends.

## 🔹 Tools Used
- Tableau
- Excel

## 🔹 Key Insights
- High attrition in Sales roles
- Younger employees (18–25) have higher attrition
- Better job satisfaction reduces attrition
